# macOS Controller Guide

## Pairing Bluetooth Controllers

1. Open **System Settings → Bluetooth**
2. Put the controller in pairing mode (see table below)
3. Click Connect when it appears
4. Verify: **System Settings → Game Controllers** — it should appear there

Up to 4 controllers can be paired simultaneously.

## Compatible Controllers

| Controller | Pairing Mode |
|---|---|
| PS5 DualSense | Hold PS + Create until light bar blinks |
| PS4 DualShock 4 | Hold PS + Share until light bar blinks |
| Xbox Series X/S | Hold pairing button until Xbox logo flashes |
| Xbox One (Bluetooth, 1708+) | Hold pairing button until Xbox logo flashes |
| Nintendo Switch Pro | Hold sync button on top |
| 8BitDo | Set to macOS/Apple mode (check 8BitDo docs) |

**Note:** Xbox 360 controllers have no Bluetooth — use them wired on the Steam Deck instead.

## Troubleshooting

- If the game doesn't detect a controller, check **System Settings → Game Controllers** first
- Try forgetting the device in Bluetooth and re-pairing
- Launch the game after pairing, not before — some controllers aren't detected if they connect after startup
